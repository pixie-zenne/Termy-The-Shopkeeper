# Termys-shop
A simple text-based game about a blind shopkeeper name Termy managing her 'Shop Of Shapes'

## Items
- Circle
- Triangle
- Square
- Rectangle
- Arrow
- Heart
- Crescent
- Star

## Characters
- Termy "The Shopkeeper"
- Yunna
- Doiko
- Tei
- Atru
- Cinni
- Bopt
- Zeec

## How to play
When prompted to enter a number, input your selected number to continue the game
